## Тестовое задание для Absolut POS, в котором представлен сервис для проведения опросов (Опросы UGC)

### Описание проекта

accounts - Приложение для работы с пользователями

core - Главные настройки

quiz - Основное приложение

Использовано: Django, DRF, PostgreSQL, psycopg2-binary, gunicorn, drf-yasg

### Задания:

#### 1. Опиши модели данных для реализации данного приложения.

Модели данных описаны в приложении quiz.models

Модель пользователей взял стандартную но только переопределил заранее, чтоб можно было вносить изменения

#### 2. Приведи пример view, которая будет возвращать вопрос, на который должен ответить пользователь в рамках опроса с id = NNN.

Пример view приведен в quiz.views.get_next_question

docker-compose up && docker-compose up -d
